
How it works:

Open the config folder and enter a version of Minecraft that has been played in the given directory.
Verify the directory location is correct.
Run the program and it will give the following
  - The assets folder of that version (hidden in the sample run)
  - The data folder of that version (hidden in the sample run)
  - ID lists for that version, including blocks and items (shown in the sample run under resources/lists)
